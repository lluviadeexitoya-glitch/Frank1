TERMUX — ACTIVAR API

¿QUÉ ES TERMUX Y POR QUÉ?

Termux es una terminal Linux que corre en tu teléfono Android. Sin instalar nada en la nube. Tú corres un mini-servidor en tu propio teléfono que actúa de puente entre la app y la API de Claude. Gratis, permanente, funciona con tu WiFi de 40 Mbps.

PASO 1 — INSTALAR TERMUX

  1. Ve a f-droid.org en Chrome
  2. Busca Termux y descarga el APK
  3. Instala (puede pedir permitir fuentes desconocidas)
  4. ⚠ NO uses el Termux de Google Play — está desactualizado

PASO 2 — CONFIGURAR TERMUX

Abre Termux y escribe estos comandos uno por uno. Toca cada línea para copiarla:

pkg update -y && pkg upgrade -y
pkg install nodejs -y
mkdir frank1proxy && cd frank1proxy
npm init -y
npm install express cors node-fetch

PASO 3 — CREAR EL PROXY

Copia y pega este comando completo en Termux de una sola vez:

cat > server.js << 'EOF' const express=require('express'); const cors=require('cors'); const app=express(); app.use(cors()); app.use(express.json({limit:'10mb'})); app.post('/api',async(req,res)=>{ try{ const r=await fetch('https://api.anthropic.com/v1/messages',{ method:'POST', headers:{'Content-Type':'application/json','x-api-key':process.env.ANTHROPIC_API_KEY,'anthropic-version':'2023-06-01'}, body:JSON.stringify(req.body) }); const d=await r.json(); res.json(d); }catch(e){res.status(500).json({error:e.message});} }); app.listen(3000,()=>console.log('FRANK1 PROXY ACTIVO :3000')); EOF

PASO 4 — TU API KEY

Reemplaza TU_API_KEY con tu clave de Anthropic. Copia este comando:

export ANTHROPIC_API_KEY=TU_API_KEY && node server.js

Tu API key la encuentras en console.anthropic.com → API Keys
